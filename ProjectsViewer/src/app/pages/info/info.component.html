I made this on my Internship in 2022. This was a project I made to learn Angular, Typescript and Asp.net.

The Offline/Online button is there so you can switch between built in data and data from the server. But I never run the server, so it's always empty. Once you click on the button it needs to refresh the page to take effect. But this can sometimes glitch out the cookies, so you might need to click it more than once to switch.

I haven't updated it much since, so excuse the styling and the lack of features. It's also not up to my programming principles anymore since I have learnt a lot since I have made this. This is not made to be a mobile website, I'll try to improve it in the future, but for now it's not a priority.

When I found out in 2023 that Github Pages existed, I decided to update it to host this website.

You can filter the projects. By hovering over the icons to see the filters. Then you can put it in the filter box. You can also ifnore filters by typing an '!' infront of the filter. The default filters are "!ignore this !pokemon essentials" to only show my programming projects by default.
